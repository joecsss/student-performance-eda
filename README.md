## Student Performance & Behavior Analysis

### Objective
Analyze student behavioral and lifestyle factors to understand their impact on academic performance.

### Tools
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

### Key Insights
- Study time and prior grades show strong correlation with final performance
- Absences negatively impact grades
- Lifestyle factors have moderate influence

### Dataset
Kaggle: Student Performance Datase
